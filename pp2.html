Computer Parts Comparison

This full-stack web application, developed in 2025, enables users to compare computer parts and laptops efficiently. Built with HTML, CSS for the front-end, and C#, SQL for the back-end, it highlights my ability to create robust, scalable solutions. Key features include:

  • Comprehensive database of computer parts with real-time updates and dynamic querying.
  • Advanced comparison tools for evaluating performance, prices, and specifications.
  • User authentication system with secure login and personalized recommendations.
  • Responsive and intuitive UI/UX optimized for all devices.
  • Efficient data management using SQL for storage and retrieval.
  • Integration of APIs for fetching real-time pricing and specs from external sources.
This project showcases my full-stack development skills, combining elegant front-end design with powerful back-end functionality.
