Flower Sales E-Commerce

This responsive e-commerce platform for flower sales was developed as a university project in 2024 using HTML, CSS, and JavaScript. It showcases my expertise in front-end development and user experience design. Key features include:

  • Intuitive product browsing with categories, filters, and search functionality.
  • Interactive shopping cart with real-time updates and local storage integration.
  • Seamless checkout process with form validation and user feedback.
  • Responsive design optimized for desktop, tablet, and mobile devices.
  • Performance optimizations for fast loading times and smooth interactions.
  • Dynamic UI elements, including hover effects and animations for an engaging user experience.
This project demonstrates my ability to create visually appealing, functional, and user-friendly web applications tailored to client needs.
